# PlannerAI

Full-stack productivity & task management platform.

**Stack:** React · TypeScript · Tailwind CSS · Recharts · Node.js · Express · Prisma · PostgreSQL (Supabase) · Google Gemini API

## Structure

- `client/` — React + TypeScript + Tailwind frontend
- `server/` — Node.js + Express + Prisma backend

## Requirements

- Node.js 24 LTS
- npm 10+
- A Supabase PostgreSQL database
